# JSON Test Spec

## How to run

## Dev

1. `npm install`
1. `npm run dev`
1. this watches index.ts and restarts when a change is made to this file.
1. this generates characters.json, episodes.json, and quotes.json for easy handoff / consumption

## Prod

1. npm install
1. npm start
1. this generates characters.json, episodes.json, and quotes.json for easy handoff / consumption
